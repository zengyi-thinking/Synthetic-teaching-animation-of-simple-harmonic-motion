# 同向同频简谐运动合成可视化

这个项目是一个交互式的物理教学工具，用于可视化同向同频简谐运动的合成效果。通过直观的图形界面，用户可以调整各种参数，观察两个简谐振动合成后的效果。

## 功能特点

- 实时可视化两个简谐振动及其合成波形
- 交互式控制：调整振幅、相位、频率等参数
- 相位差预设按钮，快速观察特定相位差的效果
- 动画控制：播放、暂停、重置
- 速度调节
- 自动计算合成波参数（振幅、相位）
- 支持视频导出（需安装 FFmpeg）

## 文件结构

项目采用模块化设计，便于维护和扩展：

- `main.py`: 主程序入口，组织各模块协同工作
- `config.py`: 配置文件，包含颜色定义、常量设置和初始参数值
- `wave_calculations.py`: 波形计算模块，实现波形数学计算
- `ui_components.py`: UI 组件模块，创建图形界面元素
- `params_controller.py`: 参数控制器，管理和更新波形参数
- `animation_controller.py`: 动画控制器，管理动画播放和更新
- `event_handlers.py`: 事件处理模块，处理用户交互事件

## 物理原理

同向同频简谐运动合成遵循以下公式：

如果两个简谐运动可以表示为：

- x? = A?・sin(ωt + φ?)
- x? = A?・sin(ωt + φ?)

则它们的合成运动为：

- x = x? + x? = A・sin(ωt + φ)

其中：

- A = √(A?? + A?? + 2・A?・A?・cos(φ? - φ?))
- φ = arctan((A?・sin(φ?) + A?・sin(φ?))/(A?・cos(φ?) + A?・cos(φ?)))

通过调整振幅和相位差，可以观察到不同的合成效果。

## 数学原理

当两个同向同频的简谐振动叠加时，满足以下关系：

x? = A?・sin(ωt + φ?)
x? = A?・sin(ωt + φ?)
x = x? + x? = A・sin(ωt + φ)

其中：

- A?, A? 为两个分振动的振幅
- ω 为共同的角频率
- φ?, φ? 为两个分振动的初相位
- A 为合成振动的振幅
- φ 为合成振动的相位

合成振动的振幅和相位可以通过以下公式计算：

φ = arctan[(A?・sin(φ?) + A?・sin(φ?))/(A?・cos(φ?) + A?・cos(φ?))]

## 物理现象观察重点

1. 当相位差为 0 时，合成振幅等于两个分振幅之和(A = A? + A?)
2. 当相位差为 π(180°)时，合成振幅等于两个分振幅之差(A = |A? - A?|)
3. 当 A? = A? 时，相位差为 π/2(90°)的合成振幅为 √2・A?
4. 合成波的相位取决于两个分振动的振幅和相位的组合
