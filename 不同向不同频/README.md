# 李萨如图形 - 不同向不同频简谐运动可视化

这个项目是一个交互式物理教学工具，用于可视化不同频率的垂直简谐运动产生的李萨如图形。通过直观的图形界面，用户可以调整各种参数，观察李萨如图形的变化。

## 功能特点

- 实时可视化两个方向的简谐振动及其合成的李萨如图形
- 交互式控制：调整振幅、频率、相位等参数
- 频率比预设按钮，快速观察经典李萨如图形
- 频率比模式选择（固定 ω1 或 ω2）
- 动画控制：播放、暂停、重置
- 可调轨迹长度
- 速度调节
- 支持视频导出（需安装 FFmpeg）

## 文件结构

项目采用模块化设计，便于维护和扩展：

- `main.py`: 主程序入口，组织各模块协同工作
- `config.py`: 配置文件，包含颜色定义、常量设置和初始参数值
- `lissajous.py`: 李萨如图形计算模块，实现相关数学计算
- `ui_components.py`: UI 组件模块，创建图形界面元素
- `params_controller.py`: 参数控制器，管理和更新波形参数
- `animation.py`: 动画控制器，管理动画播放和更新
- `event_handlers.py`: 事件处理模块，处理用户交互事件

## 物理原理

李萨如图形是由两个垂直方向、不同频率的简谐振动复合而成的图形。当两个振动的频率比为简单的有理数比时，会形成稳定的闭合曲线。

如果两个简谐运动可以表示为：

- x = A?・sin(ω?t + φ?)
- y = A?・sin(ω?t + φ?)

则李萨如图形的形状由频率比 ω?:ω? 决定：

- 当频率比为 1:1 时，形成椭圆（特殊情况下为圆）
- 当频率比为 1:2 时，形成类似 8 字形的曲线
- 当频率比为 2:3 时，形成三瓣形
- 当频率比为 3:4 时，形成四瓣形
- 其它频率比会形成更复杂的图形

相位差 φ?-φ? 决定图形的具体形态和旋转角度。
