# 同向不同频简谐运动合成可视化

这个项目是一个交互式物理教学工具，用于可视化同向但不同频率的简谐运动的合成。通过直观的图形界面，用户可以调整各种参数，观察波形的变化。

## 功能特点

- 实时可视化两个波形及其合成波形
- 交互式控制：调整振幅、频率、相位等参数
- 动画控制：播放、暂停、重置
- 速度调节
- 支持视频导出（需安装 FFmpeg）

## 文件结构

项目采用模块化设计，便于维护和扩展：

- `main.py`: 主程序入口，组织各模块协同工作
- `config.py`: 配置文件，包含颜色定义、常量设置和初始参数值
- `wave_calculations.py`: 波形计算模块，实现相关数学计算
- `ui_components.py`: UI 组件模块，创建图形界面元素
- `params_controller.py`: 参数控制器，管理和更新波形参数
- `animation_controller.py`: 动画控制器，管理动画播放和更新
- `event_handlers.py`: 事件处理模块，处理用户交互事件

## 物理原理

简谐运动是最基本的振动形式，其位移可以用正弦或余弦函数表示：

x = A・sin(ωt + φ)

其中：

- A 是振幅，决定振动的最大位移
- ω 是角频率，决定振动的快慢
- φ 是初相位，决定振动的起始位置
- t 是时间

当两个同向的简谐振动叠加时，其合成振动也是简谐的，表示为：
x = x? + x? = A?・sin(ω?t + φ?) + A?・sin(ω?t + φ?)

当两个振动的频率相同时（ω? = ω?），合成振动仍为简谐振动。
当两个振动的频率不同时，合成振动就不再是简谐的，而是周期性变化的复杂振动。

通过这个可视化工具，可以直观地观察不同参数下波形的变化，加深对简谐运动合成的理解。
